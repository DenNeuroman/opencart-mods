# ASSECURO Contact Form Fix

Модификатор для OcStore 3.x / OpenCart 3.x

## Что делает

- Исправляет заголовок `From` в письмах с формы контактов
  (решает блокировку `5.7.1 Invalid header "From"` на хостинге Ukraine.com.ua)
- Добавляет структурированное письмо менеджеру с указанием источника `[ASSECURO.COM.UA]`
- Добавляет рассылку на дополнительные адреса из настроек магазина
- Добавляет автоответ посетителю с подтверждением получения сообщения

## Совместимость

- OcStore 3.0.3.7
- OpenCart 3.x
- Тема: Chameleon (и стандартная тема)

## Установка

1. Розширення → Встановлення розширень → загрузи `assecuro_contact_fix.ocmod.xml`
2. Розширення → Модифікації → кнопка **Оновити**
3. Панель стану → Налаштування розробника → **Системний кеш** → Оновити

## Настройка дополнительных адресов

Система → Налаштування → Пошта → поле **Додаткові адреси для сповіщення**

Адреса через запятую: `sales@example.com, manager@example.com`

## Языковые файлы

Если после установки страница успешной отправки показывает `text_message` —
скопируй соответствующий файл из папки `languages/` в:
`catalog/language/uk-ua/information/contact.php`
