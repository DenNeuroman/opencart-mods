# VGRB ERP Sync

Модуль інтеграції OpenCart 3.x / OcStore 3.x з VGRB AI-ERP через API.

## Що робить

Після оформлення замовлення автоматично надсилає дані замовлення до VGRB ERP
через HTTP POST запит. Використовує Event System OpenCart — ядро не змінюється.

## Що передається

- Номер замовлення, ім'я та прізвище клієнта, телефон, email, сума
- Перелік товарів (назва, артикул, ціна, кількість)

## Встановлення

1. Розширення → Встановлення розширень → завантажити `vgrb_erp.ocmod.zip`
2. Розширення → Модифікації → **Оновити**
3. Панель стану → Системний кеш → **Оновити**
4. Розширення → Модулі → **VGRB ERP Sync** → Встановити
5. Натиснути олівець → вказати Endpoint URL → **Увімкнено** → Зберегти

## Налаштування

| Параметр | Опис | За замовчуванням |
|---|---|---|
| Статус | Увімкнути/вимкнути модуль | Вимкнено |
| URL Endpoint ERP | Адреса API VGRB | http://93.190.42.172/api/v1/orders |
| Таймаут (сек) | Час очікування відповіді | 10 |

## Логування

Всі запити (успішні і помилкові) пишуться до системного логу OpenCart:  
`Система → Журнал помилок` або файл `system/storage/logs/error.log`

## Сумісність

- OpenCart 3.x / OcStore 3.0.3.x
- PHP 7.2+
- VGRB AI-ERP v9.0
